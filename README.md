## Setup
Install with npm
```bash
npm install --save v-black-hole
```
Install with yarn
```bash
yarn add v-black-hole
```

### Vue instance
```js
import Vue from 'vue'
import VBlackHole from 'v-black-hole'

Vue.use(VBlackHole)
```
### Nuxt
Add to [modules](https://nuxtjs.org/guide/modules) section at your **nuxt.config.js**
```js
module.exports =  {
  modules:  [
    'VBlackHole/nuxt',
  ],
}
```
